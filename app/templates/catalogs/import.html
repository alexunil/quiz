Katalog importieren

Importieren Sie einen Fragenkatalog aus einer JSON-Datei.

Nur .json Dateien sind erlaubt (max. 16 MB)

JSON-Format

Die JSON-Datei muss ein Array von Fragen enthalten. Jede Frage sollte mindestens folgende Felder haben:

  • id: Eindeutige ID (String)
  • question: Fragetext (String)
  • options: Antwortmöglichkeiten (Object mit A, B, C, D)
  • correct_answer: Richtige Antwort (String oder Array)

Optional: category, subcategory, explanation, question_type
